CARS API

Domain: https://cars-api-5zai.onrender.com

USERS

URL Method Body Description Authentication
/users/myinfo GET None Returns the user information according to the sent token A valid token
/users/userslist GET None Returns the list of all users A valid administrator token
/users POST {
"name":"user",
"email":"user@gmail.com",
"password":"1234"
}		 Creates a new user None
/users/login POST {
"email":"user@gmail.com",
"password":"1234"
}		 If the user information is correct, a token is returned None
/users/user_id PUT {
"name":"user",
"email":"user@gmail.com",
"password":"1234"
}		 Updates the user data according to the ID A valid token of an administrator or of the user whose details are updated
/users/user_id DELETE None Deleted the user data according to the ID A valid token of an administrator or of the user whose details are deleted

CARS

URL Method Body Description Authentication
/cars GET None Returns all vehicles None
/cars/search/s=value_to_serach GET None Returns all vehicles whose name or information about them contains the search value None
/cars/category/category_name GET None Returns all vehicles in the requested category None
/cars/price?min=min_price&max=max_price GET None Returns all vehicles in the requested price range None
/cars/vehicle_id GET None Returns the vehicle with the requested ID none
/cars POST {
"name": "Toyota Camry",
"info": "Midsize Sedan",
"category": "Sedan",
"img_url": "your-image-url.jpg",
"price": 25000
}		 Creates a new vehicle A valid token
/cars/vehicle_id PUT {
"name": "Toyota Camry",
"info": "Midsize Sedan",
"category": "Sedan",
"img_url": "your-image-url.jpg",
"price": 25000
}		 Updates the vehicle data according to the ID A valid administrator or creator user token
/cars/vehicle_id DELETE None Deletes the vehicle data according to the ID A valid administrator or creator user token
